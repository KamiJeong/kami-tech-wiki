# Kiro 概要

Kiro は AWS が開発した AI 搭載 IDE (統合開発環境) です。VS Code を基盤として、スペック駆動開発 (SDD) ワークフローが内蔵されています。スティアリングファイル、スペックドキュメント、エージェントタスク、オートパイロットモードを通じて、構造化された仕様に基づいたAI協調開発を実現します。
