# SDD (Spec-Driven Development) 概要

Spec-Driven Development (SDD) は、コードを書く前に仕様 (Specification) を先に作成するソフトウェア開発手法です。仕様が開発チーム全体とAIエージェントの単一の情報源 (Single Source of Truth) となり、一貫性のある検証可能な成果物を実現します。
