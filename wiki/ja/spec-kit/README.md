# Spec-Kit 概要

Spec-Kit は Claude Code 環境で Spec-Driven Development (SDD) を実践するためのワークフローツールです。`specify → clarify → plan → tasks → implement → verify` の6段階パイプラインをスラッシュコマンドとして提供し、仕様作成から実装検証まで全プロセスを体系的に管理します。
