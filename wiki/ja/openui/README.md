# OpenUI

OpenUIは、自然言語プロンプトからUIコンポーネントを生成するオープンソースのAIツールです。ユーザーがテキストでUIを説明すると、OpenUIがリアルタイムでHTMLまたはReactコンポーネントを生成・レンダリングします。

このセクションの内容：

- イントロダクションとアーキテクチャ
- クイックスタートガイド
- チャットおよびGenUIインターフェース
- APIリファレンス
- システムプロンプトと仕様
