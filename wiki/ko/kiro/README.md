# Kiro 개요

Kiro는 AWS가 개발한 AI 기반 IDE(통합 개발 환경)로, VS Code 기반 위에 스펙 주도 개발(SDD) 워크플로우를 내장하고 있습니다. 스티어링 파일, 스펙 문서, 에이전트 태스크, 오토파일럿 모드를 통해 AI와 협업하여 코드를 작성합니다.
