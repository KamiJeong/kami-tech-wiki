# Spec-Kit 개요

Spec-Kit은 Claude Code 환경에서 Spec-Driven Development (SDD) 워크플로우를 실천하기 위한 도구입니다. `specify → clarify → plan → tasks → implement → verify` 6단계 파이프라인을 슬래시 명령으로 제공하여, 명세부터 구현까지의 전 과정을 체계적으로 관리합니다.
