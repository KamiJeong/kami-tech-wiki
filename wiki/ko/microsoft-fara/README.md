# Microsoft Fara 개요

Microsoft Fara는 Microsoft Research가 공개한 컴퓨터 사용 에이전트 모델 계열입니다. 텍스트 답변만 만드는 챗봇이 아니라, 브라우저 화면을 보고 클릭, 입력, 스크롤 같은 행동을 예측해 웹 작업을 수행하도록 연구된 모델입니다.
