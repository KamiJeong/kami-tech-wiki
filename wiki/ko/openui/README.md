# OpenUI

OpenUI는 자연어 프롬프트로 UI 컴포넌트를 생성하는 오픈소스 AI 도구입니다. 사용자는 원하는 UI를 텍스트로 설명하면, OpenUI가 실시간으로 HTML/React 컴포넌트를 생성하고 렌더링합니다.

이 섹션에서는 다음 내용을 다룹니다:

- OpenUI 소개 및 아키텍처
- 빠른 시작 가이드
- 채팅 및 GenUI 인터페이스
- API 레퍼런스
- 시스템 프롬프트 및 사양
