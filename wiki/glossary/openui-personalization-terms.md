# OpenUI Personalization Terminology

This multilingual glossary supports the OpenUI personalization case-reference articles.

| Concept | Korean | English | Japanese | Usage note |
|---------|--------|---------|----------|------------|
| OpenUI | OpenUI | OpenUI | OpenUI | Keep untranslated as the project/tool name. |
| Generative UI | 생성형 UI | Generative UI | 生成型 UI | UI를 미리 고정하지 않고 AI가 상황에 맞게 구성한다는 의미로 설명한다. In Japanese, pair the term with a short explanation when first introduced. |
| Real-time personalization | 실시간 개인화 | Real-time personalization | リアルタイムパーソナライズ | Use when the screen changes based on recent user context, interaction, or request. |
| Adaptive screen | 적응형 화면 | Adaptive screen | 適応型画面 | Use for a screen whose content, order, difficulty, or controls change by context. |
| Evidence status | 근거 상태 | Evidence status | 根拠ステータス | Label for whether OpenUI use is verified, partial, unclear, or only comparable. |
| AI tool | AI 도구 | AI tool | AIツール | Software that uses AI to help a user complete a task. |
| User interface | 사용자 인터페이스 | User interface | ユーザーインターフェース | The visible or interactive surface a user operates. |
| Personalization | 개인화 | Personalization | パーソナライズ | Tailoring content, ranking, layout, or workflow to a user or context. |
| Recommendation system | 추천 시스템 | Recommendation system | 推薦システム | A system that selects or ranks items a user is likely to find relevant. |
| Prompt | 프롬프트 | Prompt | プロンプト | The instruction or context given to an AI model. |
| Large language model | 대규모 언어 모델 | Large language model | 大規模言語モデル | The AI model class commonly abbreviated as LLM. |
