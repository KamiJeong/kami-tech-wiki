# Kiro Overview

Kiro is an AI-powered IDE developed by AWS, built on VS Code, with a built-in Spec-Driven Development (SDD) workflow. It uses steering files, spec documents, agent tasks, and autopilot mode to enable AI-assisted development guided by structured specifications.
