# Spec-Kit Overview

Spec-Kit is a Spec-Driven Development (SDD) workflow tool built into the Claude Code environment. It provides the `specify → clarify → plan → tasks → implement → verify` six-stage pipeline as slash commands, enabling structured management of the entire development lifecycle from specification to implementation.
