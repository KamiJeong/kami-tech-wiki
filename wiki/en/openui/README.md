# OpenUI

OpenUI is an open-source AI tool that generates UI components from natural language prompts. Users describe the UI they want in plain text, and OpenUI generates and renders HTML or React components in real time.

This section covers:

- Introduction and architecture
- Quickstart guide
- Chat and GenUI interface
- API reference
- System prompts and specification
