# OpenUI Chat GenUI

> A component library optimized for conversational chat. Every response is wrapped in a `Card`, with natural support for follow-up question patterns.

**Official Docs**: [https://www.openui.com/docs/chat/genui](https://www.openui.com/docs/chat/genui)

---

## Overview

`openuiChatLibrary` is an OpenUI component library specifically designed for chat interfaces. Unlike the general-purpose `openuiLibrary`:

- All responses are structured as a **vertical arrangement of a single Card**.
- Does not include the `Stack` component (chat is always vertical).
- Has a built-in follow-up question button (`FollowUpBlock`).

---

## Core Principles

### Chat-Specific Components

| Component | Role |
|-----------|------|
| `Card` | Default wrapper for all responses |
| `FollowUpBlock` | Follow-up question button block |
| `ListBlock` | Item list display |
| `SectionBlock` | Section divider within a response |
| `Button` | Single action button |
| `Buttons` | Button group |

### Differences from the General Library

| Aspect | `openuiLibrary` (general) | `openuiChatLibrary` (chat) |
|--------|--------------------------|---------------------------|
| Layout | `Stack` (any direction) | Always vertical card |
| Response wrapper | None | Automatic `Card` wrapping |
| Follow-up questions | None | Built-in `FollowUpBlock` |
| Use context | General UI generation | Chat interfaces |

---

## Example

OpenUI Lang generated by an LLM in chat:

```
# Search result card (openuiChatLibrary)
card = Card([
  results_section,
  follow_up
])
results_section = SectionBlock("Search Results", [
  result1,
  result2,
  result3
])
result1 = ListBlock("First Result", "Description text")
follow_up = FollowUpBlock([
  Button("Learn More"),
  Button("Search Related"),
])
```

## Comparison

| Aspect | Chat context | General app context |
|--------|-------------|---------------------|
| Library | `openuiChatLibrary` | `openuiLibrary` |
| Layout | Vertical card stack | Free layout |
| Follow-up questions | FollowUpBlock | None |
| Best for | Chat assistants | Dashboards, CRUD |
