## 各レッスンの構成

各レッスンは以下の構成になっています：

1. **学習目標** — このレッスンで学ぶこと
2. **概念の説明** — 実例を交えたコアコンセプト
3. **演習** — 実際に試せる実習課題
4. **思考質問** — 理解を深めるためのディスカッション質問
5. **次のステップ** — 関連レッスンとさらなる学習資料
